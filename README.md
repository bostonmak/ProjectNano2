This project now uses `gradle` and `JDK8`

### Building the JAR with Gradle

__To construct the jar, run:__

`./gradlew clean build` 

The jar is now created at `build/dist`

__To construct the jar and copy it to the `dist/` directory:__

Run `./gradlew buildAndCopy`

### Running the Server

Run `./gradlew run`
